# LOLA Bot 2.0 — Финальная улучшенная версия

**Поддержка и планирование для мам (и пап) особенных детей.**

Полностью доработанная версия с:

- 45+ карточек в каждой категории (для мамы + отдельные категории для пап)
- Встроенным мини-календарём
- Экспортом данных (JSON + отчёт)
- Улучшенной статистикой и графиками
- PWA (можно установить как приложение)
- Тёмной темой
- Полной поддержкой 3 языков (RU / KZ / EN)
- Вынесенными текстами в JSON

## Как запустить
1. Скачай или открой `index.html`
2. Или добавь как Web App в Telegram

## Основные разделы
- Карточки планирования
- Мини-календарь
- Трекер + заметки
- Mind (Опора + Сила)
- LOLA примеры
- Прогресс и экспорт
