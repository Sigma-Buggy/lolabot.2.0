# LOLA Bot 2.0

**Поддержка и ежедневное планирование для мам особенных детей.**

Улучшенная версия с расширенными карточками (35-45+ в каждой категории, включая для пап), мини-календарём, экспортом данных, тёмной темой, PWA и полной поддержкой 3 языков.

## Что внутри
- Карточки планирования (для мамы и папы)
- Мини-календарь с отметками
- Трекер занятий и заметок
- Mind: Опора + Сила
- LOLA примеры для ребёнка
- Экспорт данных (JSON + отчёт)
- PWA (установка как приложение)
- Тёмная тема
- Полная многоязычность (RU / KZ / EN)

## Запуск
Открой `index.html` в браузере или добавь как Telegram Web App.

## Структура
- `index.html` — основной файл
- `i18n/` — переводы в JSON
- `data/cards.json` — карточки
- `manifest.json` — PWA
