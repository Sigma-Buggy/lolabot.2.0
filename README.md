# LOLA Bot 2.0

**Поддержка и ежедневное планирование для мам особенных детей.**

Улучшенная версия оригинального lola-app с фокусом на **карточки планирования** (35-45+ в каждой категории).

## Основные улучшения
- Разделённый код (HTML/CSS/JS + data)
- Расширенная библиотека карточек
- README, PWA-ready
- Легко добавлять новый контент

## Как использовать
Открыть `index.html` или задеплоить как Telegram Web App.
